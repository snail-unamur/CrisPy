# Privilégiez la méthode join() (String Join Method)

## Pourquoi ?

Utiliser l'opérateur `+` pour concaténer des chaînes dans une boucle crée un nouvel objet chaîne à chaque itération, ce qui engendre une multiplication de la consommation mémoire et des allocations inutiles.

## Mauvaise pratique

```python
mots = ["Python", "est", "génial"]
phrase = ""
for mot in mots:
    phrase += mot + " "
print(phrase)
```

Explication : Cette approche crée une nouvelle chaîne à chaque itération, conduisant à une complexité O(n²) en termes de mémoire.

## Bonne pratique

```python
mots = ["Python", "est", "génial"]
phrase = " ".join(mots)
print(phrase)
```

Explication : La méthode join() effectue une seule allocation de mémoire, offrant une complexité O(n) et une performance optimale.

## Performance

L'utilisation de `+` dans une boucle pour concaténer des chaînes est un "anti-pattern". Chaque concaténation recopie toute la chaîne en mémoire, ce qui devient dramatique pour de grandes listes (par exemple, 10 000 mots génèrent environ 100 millions d'allocations).
